# 🎖️ Tanks 2D

Проект двухмерной аркадной игры про танковые сражения.

---

## 🎮 Описание геймплея

Игрок управляет боевым танком и выполняет боевые задачи на замкнутых локациях (уровнях):
* **Система уровней:** Игра разделена на этапы. После уничтожения всех вражеских ботов на первом уровне (`Level1`) осуществляется переход на промежуточную сцену победы `WinScene`. Уничтожение всех противников на втором уровне (`Level2`) ведет к финальному экрану `FinalScene`.
* **Искусственный интеллект:** Противники (боты) активно патрулируют карту, выслеживают игрока и ведут по нему огонь.
* **Интерфейс и эффекты:** На экране отображается динамический счетчик здоровья в формате `текущее/максимальное`. Уничтожение как ботов, так и танка игрока сопровождается эффектом покадровой анимации взрыва. При потере очков здоровья до нуля игра автоматически перезапускает текущий уровень.

---

## 🕹️ Управление

* **`W` / `S` (или стрелки Вверх/Вниз)** — Движение танка вперед и назад.
* **`A` / `D` (или стрелки Влево/Вправо)** — Поворот корпуса / башни танка.
* **Левая Кнопка Мыши** — Выстрел основным орудием.

## 🚀 Инструкция по запуску

Проект разработан на игровом движке **Unity**. Чтобы запустить игру в редакторе, следуйте инструкции:

1. **Скачайте проект:** Склонируйте репозиторий или скачайте архив с исходным кодом (папку проекта) на свой компьютер.
2. **Откройте Unity Hub:** Запустите программу *Unity Hub* на вашем ПК.
3. **Добавьте проект:** В правом верхнем углу нажмите кнопку **`Add`** затем `Add project from disk`.
4. **Выберите папку:** В открывшемся проводнике выберите скачанную папку проекта (в которой находятся папки `Assets`, `ProjectSettings` и т.д.).
5. **Запуск:** Дождитесь импорта файлов, откройте стартовую сцену в папке `Assets/Scenes` и нажмите кнопку **Play** в верхней части редактора.
